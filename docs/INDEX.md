## 📞 Suporte Rápido

| Problema | Solução |
| ---------- | --------- |
| App não inicia | [QUICKSTART.md§troubleshooting](QUICKSTART.md) |
| Porta ocupada | `lsof -i :3000` ou `:5173` |
| Dados não aparecem | Verifique `backend/clientes.json` |
| Tema não funciona | Limpe localStorage do navegador |
| Código não compila | Execute `bun install` novamente |
